# Requirements
- dotnet: 
```
sudo apt update
sudo apt install dotnet-sdk-6.0
```
---
# Copy repository
```
git clone -b np_project https://github.com/KarolHartwig/SharedData.git
```
---
# Build in repo
```
cd SharedData/c#/
dotnet build HangMan.sln
```
---
# Tests
```
cd SharedData/c#/
dotnet test HangMan.sln
```
---
# Run
```
cd SharedData/c#/
dotnet run
```
